# üé¨ Moovly - Modern Movie Catalog App

**Moovly** adalah aplikasi katalog film modern berbasis web yang dibangun menggunakan **React.js**. Aplikasi ini memungkinkan pengguna untuk menjelajahi film-film terbaru, mencari judul spesifik, dan memfilter berdasarkan kategori genre secara real-time menggunakan data dari API **The Movie Database (TMDB)**.

Proyek ini disusun untuk memenuhi tugas akhir (UAS) mata kuliah **Web Development**.

---

## ‚ú® Fitur Utama

- **Real-time Data Fetching**: Sinkronisasi data film terbaru dan trending langsung dari API TMDB.
- **Dynamic Search Bar**: Pencarian film yang responsif dengan fitur *debounce* untuk efisiensi performa.
- **Genre Categorization**: Filter film berdasarkan kategori (Action, Comedy, Drama, Horror, dll).
- **Dual Theme Support**: Mendukung Mode Gelap (Dark Mode) dan Mode Terang (Light Mode) yang nyaman di mata.
- **Responsive Design**: Antarmuka yang dioptimalkan untuk berbagai ukuran layar (Desktop, Tablet, dan Smartphone).
- **Glassmorphism UI**: Desain modern dengan efek transparansi dan blur pada elemen navigasi.
